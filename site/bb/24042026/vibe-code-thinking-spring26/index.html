The Take

There is ALWAYS a trade-off for using technology. Yes AI can be bad, but so can all tech.

Paul Johnson had an incredible thread on Bluesky that puts words to how I feel about AI, value and the constant tradeoff: https://bsky.app/profile/pdj.bsky.social/post/3mjz5kl3qvk2i

He also takes the time to point out that ALL of the internet content, and most software now already relies on AI for code. Github maintainers have had to adapt how they think about their roles and the FLOOD of AI code being sent their way.
https://steve-yegge.medium.com/vibe-maintainer-a2273a841040

If you haven't already - make sure you are following Gene Kim and Steve Yegge as they continue to lead the way in Vibe Coding (they wrote the book (literally)).
Gene gave a GREAT talk recently where he pointed out dev roles that will literally no longer exist. Probably a good exercise for any professional!
